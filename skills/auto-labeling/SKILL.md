## SAM 2 vs SAM 3: Selection Guide

Our unified Ultralytics setup supports dropping in weights for both SAM 2 and SAM 3. Here is when to use which:

### SAM 2 (Segment Anything 2)
**Core Capability**: Real-time object and video masking from spatial hints.
*   **The "Speedy Sculptor"**: Needs a starting point (like a Bounding Box or a Point Click) from you or another model.
*   **When to use**: 
    - You already have object bounding boxes (e.g., generated by an upstream VLM or YOLO).
    - You need incredibly fast, low-compute mask refinement on tens of thousands of images.
    - You are processing video frames and need to track a *specific* targeted object across time.
*   **Recommendation**: Use `sam2_b.pt` or `mobile_sam.pt` for the most lightweight mask extraction.

### SAM 3 (Segment Anything 3)
**Core Capability**: Open-Vocabulary "Concept" Segmentation.
*   **The "Autonomous Search Engine"**: Given merely a text prompt (e.g., "all the cracked windshields") or an image example, it will scan the image/video, detect **all** matching instances, and mask them. Unifies Detection, Tracking, and Segmentation.
*   **When to use**:
    - You want a completely **Zero-Shot Automatic Labeling** pipeline from an empty image directly to masks using natural language.
    - You are dealing with complex concepts without a pre-existing Bounding Box detector.
*   **Recommendation**: Use when maximum automation is preferred over processing speed/GPU capability.

## Workflow 1: Precision Masking with SAM

Convert rough bounding boxes or points into pixel-perfect masks.

### Step 1: Load Initial Data and Run SAM

If you have pre-existing rough labels or grid points, you can use SAM to refine them.

```bash
# Using SAM 2 from Ultralytics
python scripts/sam_segmenter.py data/raw/pre_labels.json \
    --model sam2_b.pt \
    --output segment_labels.json
```

## Workflow 2: Formatting & Human-in-the-loop

Automated labels often require refinement or conversion to standard formats.

### Step 1: Export to Review Tool

```bash
# Export to Label Studio for final human verification
python scripts/label_converter.py data/ \
    --to label_studio \
    --output review_project/
```

### Step 2: Export Data for Model Training

```bash
# Convert JSON labels to YOLO format
python scripts/label_converter.py data/ \
    --to yolo \
    --output dataset_yolov8/
```
